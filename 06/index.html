A forma implícita

Quando escrevemos z = f(x,y) = xy, estamos usando a chamada forma explícita de f , pois a regra está pronta para ser usada; basta substituir os valores de x e y. Mas existe outra maneira de a escrevermos, também muito útil: a forma implícita:

φ(x,y,z) = 0.

Para obter a forma implícita, simplesmente reorganizamos z = f(x,y) de modo que no primeiro membro da igualdade tenhamos x,y,z, e no segundo, zero (o zero é importante). Como você acha que deve ser a dorma implícita de z = f(x,y) = xy?

Vejamos um exemplo de como esta forma pode ser útil. Para isso, deixemos de lado f e recordemos a equação da reta na geometria analítica: y = ax + c. a é o coeficiente angular e b, o coeficiente linear. Isto funciona muito bem desde que a reta não seja vertical (veja figura novamente). Embora ainda seja uma reta, y não é mais uma função, pois um mesmo x resulta em y diferentes. A situação é análoga a dizer que nosso retângulo de lados x e y tem, ao mesmo tempo, áreas z1 e z2. Ainda assim é uma reta, e a forma implícita dela resolve o problema: ax + by + c = 0. Tente identificar a,b e c para a reta acima (dica: comece escrevendo x = 1).

φ(x,y,z) = x + y + = 0

A resposta correta é φ(x,y,z) = x - 1 = 0. Isto é, a = 1, b = 0 e c = -1. Sua resposta: φ(x,y,z) = x + y + = 0

Como você pode ver, a forma implícita da reta não tem a mesma limitação da forma explícita. Esta é apenas uma das vantagens. Nos próximos tópicos você verá outras utilidades dela.
